Heroku supports [auto_explain](https://pganalyze.com/docs/explain/setup/auto_explain) to collect
EXPLAIN plans automatically on [most plans](https://devcenter.heroku.com/articles/heroku-postgres-settings#auto-explain).

```sh
heroku pg:settings:auto-explain on -a [your-collector-app-name]
```

Once Log Insights is working, enabling EXPLAIN plan collection just requires turning on the collector setting. You
can do this by setting a config var on your Heroku app:

```
$ heroku config:set PGA_ENABLE_LOG_EXPLAIN=true -a [your-collector-app-name]
```

