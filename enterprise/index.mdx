pganalyze Enterprise Server is a fully self-contained version of pganalyze that runs
behind your firewall. It is designed for environments where sensitive data must stay
within your own infrastructure — financial services, healthcare, air-gapped networks, or
other situations where our cloud-hosted service is not a good fit.

pganalyze Enterprise Server gives you the same query performance monitoring, index and
query advisor, EXPLAIN plan analysis, and log-based insights as the hosted service, deployed
entirely within your environment.

If you'd like to start a trial of pganalyze Enterprise Server,
**[please contact us](/contact)**.
