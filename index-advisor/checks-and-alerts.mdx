Index Advisor checks are enabled by default for all your databases, but you need
to set up an alert rule and destination (for example, a Slack channel) to be
notified of newly detected missing or unused indexes.

Index Advisor supports the following checks:
 - [Missing indexes](/docs/checks/index_advisor/indexing_engine)
 - [Unused indexes](/docs/checks/schema/index_unused)

You can read more details about the [Alerts & Check-Up system](/docs/checks) in
its own section of the documentation.
