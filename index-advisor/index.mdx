The pganalyze Index Advisor is a tool to analyze your queries, determine how Postgres
will scan the tables involved, and suggest indexes to make those scans more efficient.

This section describes the in-app Index Advisor. Looking for the standalone
Index Advisor documentation? [Continue here.](/docs/index-advisor/standalone)

 * [Introduction](/docs/index-advisor/introduction)
 * [Navigating Index Advisor](/docs/index-advisor/navigating)
 * [How to Reason About Recommendations](/docs/index-advisor/reason-about-recommendations)
 * [Configuring Checks and Alerts](/docs/index-advisor/checks-and-alerts)
 * [How to Test Recommendations](/docs/index-advisor/test-recommendations)

To understand better how recommendations are made, view the documentation for the [pganalyze Indexing Engine](/docs/indexing-engine), which runs behind the scenes and provides the data for the Index Advisor.
