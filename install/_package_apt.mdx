Run the following commands as root on your server:

```
curl -L https://packages.pganalyze.com/pganalyze_signing_key.asc | apt-key add -
echo "deb [arch=amd64] https://packages.pganalyze.com/ubuntu/xenial/ stable main" > /etc/apt/sources.list.d/pganalyze_collector.list
apt-get update
apt-get install pganalyze-collector
```

**The above configuration is for Ubuntu 16.04 LTS.**

For Ubuntu 14.04 LTS replace "xenial" with "trusty". For Debian replace "ubuntu/xenial" with "debian/jessie" or "debian/stretch".

These repositories are updated by pganalyze whenever a new stable version of the
collector is released.
