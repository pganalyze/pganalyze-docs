The following command creates a new IAM policy named pganalyze:

```bash
# First, download the policy JSON file
curl -O https://pganalyze.com/docs/install/amazon_rds/pganalyze-collector-policy.json

# Update file://pganalyze-collector-policy.json to point to the downloaded file
aws iam create-policy \
    --policy-name pganalyze \
    --policy-document file://pganalyze-collector-policy.json
    --description "Allow the pganalyze collector to access RDS information"
```
