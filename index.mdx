Learn more about how pganalyze works, and how to connect it with your Postgres database. pganalyze enables you to deliver consistent PostgreSQL performance and availability, at any scale.

## Installation guides

In order to use pganalyze you need to install the pganalyze collector in your environment, either on the database server directly, or a container / VM close to it when using managed database services. The collector sends query data and other statistics on a continuous basis â€” to the pganalyze cloud service, or to your pganalyze Enterprise Server installation.
